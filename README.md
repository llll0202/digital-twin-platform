# 钢铁厂数字孪生平台

基于 Vue3、TypeScript、Three.js、ECharts 和 Express 搭建的钢铁厂数字孪生可视化平台，包含三维厂区展示、设备状态面板、仪表数据展示和 AI 监测助手功能。

## 主要功能

- 钢铁厂三维场景展示
- 设备状态数据可视化
- 温度、压力等仪表盘展示
- AI 监测助手面板
- Express 后端接口模拟设备风险分析结果

## 技术栈

- Vue3
- TypeScript
- Vite
- Three.js
- ECharts
- Express
- Less / SCSS

## 启动项目

前端：

```bash
npm install
npm run dev
```
